## 议题详情

### 1. 商品内容生成工具功能演示

| 功能点      | 说明                                         |
| :------- | :----------------------------------------- |
| **基本流程** | 上传商家素材图片或 Word 文档 → 系统自动分析 → 填充左侧配置区       |
| **内容风格** | 默认学习用户文风和笔记风格；右侧提供小红书种草、趣味风格选项；生成时提供三版风格选择 |
| **编辑功能** | 支持预览效果、拖拽图片、文字编辑                           |
| **后续进化** | 简化为一列界面，下方可添加"整体扩充文案"等标签功能                 |


### 2. 比价功能讨论

| 项目 | 内容 |
|:------|:------|
| **技术难点** | 各大平台反爬机制严格，自动比价实现较难 |
| **当前方案** | 人工截取竞品价格图片（如淘宝截图），圈出价格对比信息后上传 |
| **示例场景** | 自家产品卖 ¥10，竞品卖 ¥16 → 截取竞品价格图并标注 |
| **自动生成** | 暂不支持自动生成比价图，仅能以文字列表形式呈现 |

### 3. AI 图片生成功能讨论

| 项目 | 内容 |
|:------|:------|
| **能力演示** | 可根据一张牛奶图生成不同使用场景的宣传图 |
| **真实性问题** | 目前 AI 生成的文字可能存在火星文问题 |
| **批量需求** | 需 360 度展示商品各个角度的图片 |
| **当前方案** | 先拍摄商品 360 度照片，再由 AI 优化生成 |
| **结论** | 图片生成功能**暂不集成**到当前工具 |

### 4. 商品信息配置讨论

**必填项 vs 非必填项：**

| 信息项 | 必要性 | 备注 |
|:--------|:--------|:------|
| 核心配料原料 | **必需** | |
| 净含量 | **必需** | |
| 售价 | **必需** | |
| 保质期 | **必需** | |
| 产地 | 非必需 | |
| 发货地 | 非必需 | 由供应商直接提供 |
| 快递公司 | 非必需 | 由供应商直接提供 |
| 物流售后 | 非必需 | 由供应商直接提供 |

**文案结构优化：**

| 优化要点 | 说明 |
|:----------|:------|
| **价格排序** | 价格信息需靠前展示 |
| **卖点突出** | 文案重点突出产品卖点，物流售后信息单独列示 |
| **模块调整** | 支持手动调整模块顺序 |
| **使用场景** | 可简略，非重点内容 |
| **AI 提炼卖点** | 可从供应商信息中自动识别和提炼商品卖点 |

**文案结构顺序（可手动调整）：**

> **钩子** → **口感体验** → **原料溯源** → **成分科普** → **品牌背书**
